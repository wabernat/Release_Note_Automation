Ticket Severity Component Ticket Description

MD-712

Critical

Metadata

When a bucket contains a large number (10K or more) of delete markers (especially if these delete markers are consecutive), listing objects in the bucket can timeout due to the need to skip many of delete markers without any listing output.

Workaround: Keep the number of delete markers low, by regularly cleaning older versions from the bucket.

RING-31615

Critical

dewpoint, geosync

The temporary journal folder of the passive connector server is removed by systemd, which results in a journal not generating.

Workaround: Launch the following command on the source site's connector servers:

echo "x /var/tmp/geosync" >> /usr/lib/tmpfiles.d/scality-dewpoint-fcgi.conf
