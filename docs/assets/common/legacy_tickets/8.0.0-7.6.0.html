This section lists fixes and enhancements to RING 8.0.0 and S3 Connector version 7.5.0.

Tickets listed in the following table were fixed or enhanced since RING version 7.4.4.7.

RING-30445

Critical

dewpoint, geosync, setup

The dewpoint-sofs.js config files can be silently archived as dewpoint-sofs.js-0, -1, -2, etc. and replaced with similar files in which journaling is not enabled. When this issue occurs, all seems to be in normal working order from the user’s perspective, and nothing is broken, but journal entries are no longer written (and are therefore not replicated). The geosync feature does not immediately stop as it is still playing the backlog of journal entries and file transfers, however at some point it will catch up and subsequently stall.

Workaround: Each time an action is performed on dewpoint connectors or after a connector upgrade, check that the /etc/dewpoint* files (dewpoint-sofs.js and dewpoint.js) are as expected.
