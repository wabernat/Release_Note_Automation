RING 8.0.0.2

Ability to Defer the Writes of Parts in an Erasure Coding Write Scenario

The write operation can be completed as soon as the minimum number of part(s) specified has been written to the RING. This reduces and possibly nullifies the impact of some of the slowest partial write operations on the overall operation.
