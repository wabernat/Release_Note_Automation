Supported Operating Environments

Scality RING 8 is supported on the following operating system distributions (64-bit only) for storage servers and connectors:

RING with S3 Connector RHEL 7.x / CentOS 7.x
RING only RHEL 6.x / CentOS 6.x (support not currently available for the Scality Installer, the Full Geosynchronization feature, the File Access Coordination feature, or S3 Connector)
