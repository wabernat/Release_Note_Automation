Scality RING 8 consists of two distinct components, the RING and the S3 Connector. Because these components are on separate development paths, each RING 8 release may embed version increments of each that differ. The initial release of RING 8, for example, comprised RING v8.0.0.0 and S3 Connector v7.5.0.0.

Product Organization

Scality RING Core, Installation, Supervision and Monitoring

  • All core RING features, including the full scale-out architecture, self-healing, flexible data protection policies with replication and Advanced Resiliency Configuration (ARC) erasure coding. It also includes RING core improvements that enable better performance, scalability and maintainability.
  • The Scality RING Installer and its set of pre-install and post-install check tools. Its hardware inventory enables streamlining typical RING installations on known x86 hardware.
  • The RingSH command line interface, the Supervisor UI, and the set of RESTFul APIs for Scality RING management and monitoring (cloud or local).
  • Support for single-site and multiple (2 or 3) site geo-distributed deployments for file and object connectors.
